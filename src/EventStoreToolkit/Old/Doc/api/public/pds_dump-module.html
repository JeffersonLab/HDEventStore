Function Details

decodeVersionInfo(fileName)

Decode VersionInfo from beginrun record. VersionInfo consists of:
  • softwareRelease : string
  • specificVersionName : string
  • configurationHash : string
  • ordinal : unsigned int no packing
  • ancestors : container of string
So, it alwasy grow, e.g. post-p2 file will contains two VersionInfo's, one for itself and one for it's parent. So, the underlying algorithm creates a list of VersionInfo's in the following format: [(childTag,[softRel,svName,hash,id,parent1,parent2,...]),(parentTag,[...])] This method returns a (svName,[listOfParents])
